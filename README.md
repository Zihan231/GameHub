## 🧩 About

**GameHub** is a React-based gaming portal designed for modern web users.  
It brings together popular games with an elegant interface, fast performance, and responsive design — giving players an all-in-one entertainment experience.  

Whether you're here to explore trending titles or simply relax with casual games, **GameHub** has you covered.  

---

## 🚀 Features

✨ **Modern UI/UX** – Built with TailwindCSS and DaisyUI for a premium look.  
🎮 **Game Library** – Explore a growing collection of interactive browser games.  
⚡ **Fast & Responsive** – Optimized for all devices using React and Firebase Hosting.  
🔐 **User Accounts** – Firebase Authentication for sign-up and login.  
🌙 **Dynamic Themes** – Smooth dark/light mode integration (if enabled).  
🧭 **Smooth Navigation** – Page routing handled seamlessly with React Router.  
📱 **Cross-Platform Support** – Works perfectly on desktop, tablet, and mobile.

## 💡 Usage

1. Visit the website: [GameHub](https://gamehub-d3a9d.web.app/)  
2. Browse through available games on the homepage.  
3. Click on a game card to see details and start playing.  
4. (Optional) Log in or register for a personalized experience.  
5. Enjoy endless gaming fun!
